## 🌟 Featured Projects

### 🏥 [VillageCare - Village Management Portal](https://github.com/Chamudu/VillageCare) ⭐ FLAGSHIP PROJECT

> **A comprehensive administrative platform for Grama Niladhari (GN) Officers to manage village operations in Sri Lanka**

**Live Demo:** [village-care-mu.vercel.app](https://village-care-mu.vercel.app)

<div align="center">

![Commits](https://img.shields.io/badge/Commits-100%2B-success?style=flat-square)
![Branches](https://img.shields.io/badge/Branches-13-blue?style=flat-square)
![Status](https://img.shields.io/badge/Status-Production-brightgreen?style=flat-square)
![Year](https://img.shields.io/badge/Year-2025-orange?style=flat-square)

</div>

**🎯 Key Features:**
- 📊 **Dashboard Analytics** - Real-time statistics on residents, households, and certificates
- 👥 **Resident Management** - Complete resident registration and profile management system  
  - Self-registration with NIC validation (both old & new formats)
  - Auto-fill DOB and gender from NIC
  - Document upload system (NIC, birth certificate, address proof)
  - Registration request approval workflow
- 📜 **Certificate Processing** - Handle income certificates, residence certificates, and other official documents
- 🏠 **Household Tracking** - Manage household information and family structures with economic data
- 📢 **Announcements System** - Create and manage village-wide announcements
- 🔔 **Notification Center** - Real-time alerts for certificate requests and registrations
- 📍 **Location & Status Tracking**  
  - GN officer real-time location tracking with Leaflet maps
  - Status updates (Working/On Break/On Leave)
  - Office/Home/DS location configuration
  - Google Maps integration for villagers
- 💬 **Complaint Management System**  
  - Villagers can submit complaints with categories and priorities
  - GN officers can review, respond, and resolve complaints
  - Comprehensive reporting and tracking
- 🔐 **Role-based Access Control** - Secure access for GN, DS, and Villager roles
- 🎨 **Premium UI/UX**  
  - Custom loading screen with glassmorphism design
  - Responsive design for mobile and desktop
  - White-themed professional interface

**🏗️ Technical Architecture:**
- **Backend:** PHP MVC Architecture with custom framework
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Database:** MySQL with comprehensive schema
- **Maps:** Leaflet.js for interactive maps, Google Maps API
- **Security:** Session management, CSRF protection, prepared statements
- **Deployment:** Vercel with environment-specific configuration

**📂 Project Structure:**
- 13 Active Branches (Gn-backend, villager, DS, dev, integration branches)
- 100+ commits documenting complete development journey
- Comprehensive documentation and SQL migrations
- Modular component architecture

**🚀 Recent Major Updates (2025-2026):**
- Implemented location tracking with GPS coordinates
- Added complaint management system
- Built villager self-registration with document uploads
- Created registration approval workflow for GN officers
- Deployed to production on Vercel
- Added loading screens and performance optimizations
